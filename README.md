# 🩸 Kakuja FM // Tokyo Ghoul Audio Player

Um player de música web minimalista e cirúrgico inspirado na estética dark/neon de **Tokyo Ghoul**. O projeto traz uma combinação agressiva das cores preto profundo, branco puro e vermelho sangue, contando com detalhes visuais marcantes como a icônica lírio-aranha vermelha animada (*Higanbana*) e referências ao universo dos ghouls.

---

## 🎧 Demonstração Visual & Estética
- **Paleta de Cores:** Totalmente personalizada em tons de `#000000` (Preto), `#ffffff` (Branco) e `#d60007` (Vermelho Sangue).
- **Identidade:** Nome inspirado nos ghouls mais poderosos (*Kakuja*), com sessões divididas pelos distritos e facções do anime (Anteiku, Aogiri Tree, Laboratório).
- **Lírio-Aranha:** Um vetor exclusivo posicionado no menu lateral que pulsa dinamicamente via CSS Keyframes, simbolizando a aceitação do lado ghoul.
- **Cards Flutuantes:** Layout dinâmico com cards que se inclinam e ganham brilho neon vermelho ao passar o mouse.

## 🛠️ Tecnologias Utilizadas
O projeto foi desenvolvido do zero utilizando apenas tecnologias nativas da web (Vanilla Architecture):
- **HTML5:** Estruturação semântica da aplicação e uso da API nativa de `<audio>`.
- **CSS3:** Estilização cirúrgica, filtros de brilho (*drop-shadow*), gradientes lineares/radiais e animações de pulso.
- **JavaScript (ES6):** Motor de áudio dinâmico, lógica de controle de faixas (Play/Pause/
